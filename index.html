Control Stock API

Test API con datos para Sistema de Control de Stock

Prueba

                    fetch('https://apicontrolstock.azurewebsites.net/api/producto/16')
                        .then(response => response.json())
                        .then(json => console.log(json))
                
                    {}

Rutas

/api/producto 12 Productos
/api/categoria 2 Categorias (Hardware y Software)
/api/marca 6 marcas (3 de Hardware y 3 de Software)
/api/usuario 3 Usuarios

Endpoints

GET api/producto
GET api/producto/{id}
GET api/producto/categoria/{nombreCategoria}
GET api/producto/marca/{nombreMarca}
POST api/producto
PUT api/producto/{id}
DELETE api/producto/{id}
